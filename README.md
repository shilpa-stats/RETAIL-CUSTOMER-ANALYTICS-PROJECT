# Retail Customer Analytics Project | Data Analysis Case Study

## Project Overview

This project analyzes retail transaction data to identify customer purchasing behavior and generate business insights.

## Business Problem

Retail companies need to understand:

Customer segments
Purchase behavior
High value customers
Sales trends

This project analyzes transaction data to support business decision making.
